Бот для дискорда, с функциями:
1. Автовыдача ролей
2. Выдача роли по эмодзи
3. Музыка(vk, youtube)

Библиотеки:
 - Discord.NET (весь проект)
 - Victoria (музыка)
 - Microsoft(DI, Hosting, etc.)

Также используется сборка Lavalink v4 для сервера музыки.

```
Префикс-команды
!hello - ответ от бота
!embed - тестовый эмбед
!clear - требует управление сообщениями(очищает все сообщения в чате в диапазоне 7 дней)
!random - % of gay
!help или (!h) - узнать команды бота

Музыкальные команды
!play или (!p) "ссылка"
!join или (!j) - пригласить бота если сразу не зашёл
!leave или (!l) - бот покидает канал
!stop или (!s) - закончить музыку
!pause - пауза
!resume - продолжить
!next или (!n) - пропустить
!repeat "режим" или (!r "режим") - повтор треков(0-ничего, 1 - текущий, 2 - все в очереди)
!volume "значение" - изменить громкость
!playlist или (!list) - просмотреть очередь треков

Команды для ролей:
!rr help - помощь по настройке создания ролей
!rr createpanel - создать сообщение для постановки там смайликов
!rr add "id-сообщения" "смайлик" "роль" - добавить смайлик для получения роли
!rr remove "id-сообщения" "смайлик" - убрать привязку роли к смайлику
!rr info - показать общее количество привязок
!rr list - показать все привязки
!set-role "пользователь" "роль"
!remove-role "пользователь" "роль"

Команды для автовыдачи роли:
!ar_switch - Включить/выключить систему
!ar_add @Новичок  - Добавить роль "Новичок"
!ar_remove @Новичок  - Удалить роль "Новичок"
!ar_status - Показать текущие настройки
!ar_help - Показать справку

Слэш-команды:
/test - эмбед(плашка)
/parameter - эмбед с текстом

```
